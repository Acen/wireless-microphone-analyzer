Wireless Microphone Analyzer

The Wireless Microphone Analyzer shows the frequency spectrum captured via the RF Explorer hardware:

Wireless microphone vendors have defined specific frequency ranges for their equipment which they gave proprietary names. This app allows you to preselect these vendor specific bands. E.g. if you bought a "Sennheiser WE 100 G3" in Sennheiser's E-Band, you can select this frequency range to check whether there are any interferences. You can also overlay the vendor recommended channel frequency presets, which are optimized by vendors to guarantee an intermodulation free operation of multiple microphones in parallel. The app also shows country specific forbidden frequency ranges in red, depending on the selected country.

Connection

When starting the app, it automatically tries to detect the serial port to which the RF Explorer is attached to. If this doesn't work, open the "Port" menu and select the corresponding port manually. In case there is still no display, please restart the app or press <CTRL><R>. It looks like sometimes the serial ports are not detected properly by the underlying Electron framework.

UI Description

The real time graph works in “peak hold” mode. That means that the highest level detected for a certain frequency is kept. There is also a red line at -85dBbm which commonly seems to be a maximum noise level which can be tolerated for wireless microphones.

It is possible to show/hide each of the following graph overlays:

  • Recommended manufacturer channels
  • Forbidden ranges
  • Congested / forbidden channels

by clicking the corresponding entry in the legend on top.

Once a channel preset for a certain vendor has been selected via the menu, the bank/channel number and also the corresponding frequency is displayed on top for that corresponding channel.
